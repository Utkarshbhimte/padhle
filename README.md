# Padhle (पढ़ले)\n\nAI-powered reading comprehension quiz platform for Indian students.\n\n> "Padhle beta" — every Indian parent, ever.\n\n## Features\n- Upload textbook screenshots → AI generates quizzes\n- Dual-pane quiz experience (passage + questions)\n- AI evaluation with gentle grammar feedback\n- Adaptive difficulty based on past performance\n- Netflix-style profile picker (parent + children)\n- Duolingo-inspired UI with streaks, XP, mascots\n- ICSE syllabus support (book + chapter linked)\n\n## Tech Stack\n- Next.js 14 (App Router, TypeScript)\n- Tailwind CSS\n- Prisma + PostgreSQL\n- Zustand\n- NextAuth.js\n\n## Getting Started\n\n```bash\nnpm install\nnpm run dev\n```
